…o crear un nuevo repositorio en la línea de comandos
    echo "# PIAD-302-TRUJILLO" >> README.md 
    git init 
    git add README.md 
    git commit -m "primer commit" 
    git branch -M main 
    git remote add origin https://github.com/2000JEFFERSON/PIAD-302-TRUJILLO.git
    git push -u origin main

…o enviar un repositorio existente desde la línea de comandos
    git remoto agregar origen https://github.com/2000JEFFERSON/PIAD-302-TRUJILLO.git
    git rama -M principal 
    git push -u origen principal
